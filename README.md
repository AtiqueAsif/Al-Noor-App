# Al-Noor – Islamic Lifestyle & Spiritual Companion App UI

Al-Noor is a modern Islamic lifestyle mobile application designed to support users in their daily spiritual practices, including prayer tracking, Quran reading, dua, and dhikr (tasbih).

This project presents a complete UI/UX design of a spiritually focused digital product that combines religious guidance with modern user interface design principles.

The interface was designed using Figma and supported by Google Stitch, with a strong emphasis on clarity, calm visual aesthetics, and meaningful user experience.

---

## Project Overview

Many users seek a structured and engaging way to maintain consistency in their daily ibadat (worship) and spiritual habits.

Al-Noor is designed as a centralized platform that helps users:

- Track daily prayers  
- Read and reflect on the Quran  
- Access authentic duas and hadith  
- Maintain tasbih (dhikr) counts  
- Find Qibla direction accurately  
- Build consistent spiritual routines  

The goal of this project is to create a simple, intuitive, and spiritually engaging mobile interface that enhances daily Islamic practices.

---

## Design Concept

The design focuses on creating a calm and distraction-free environment that aligns with spiritual usage.

Key design considerations include:

- Minimal and clean layout  
- Soft visual hierarchy for readability  
- Focus on daily habit tracking  
- Integration of religious content with modern UI  
- Mobile-first user experience

## Core Features

### Home Dashboard

- Daily prayer overview  
- Next prayer timing  
- Spiritual progress tracking  
- Daily ibadat completion status  
- Hadith of the day  

### Prayer System

- Accurate prayer times  
- Daily prayer schedule  
- Location-based adjustments  

### Quran Module

- Surah reading interface  
- Arabic text with translation  
- Verse-by-verse readability  

### Dua Collection

- Authentic duas with translation  
- Context-based supplications  
- Audio recitation support  

### Tasbih (Dhikr Tracker)

- Customizable tasbih goals  
- Real-time counting system  
- Daily streak tracking  
- Total dhikr history  

### Qibla Finder

- Direction detection using compass  
- Real-time orientation  
- Calibration guidance for accuracy  

### Progress Tracking

- Daily ibadat completion percentage  
- Prayer count tracking  
- Streak system for consistency  

### Profile & Settings

- User profile and activity stats  
- Theme customization (light/dark mode)  
- Notification controls  
- Prayer and reminder settings
